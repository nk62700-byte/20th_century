# 20th_century
This repository was created as part of an exercise to practice using Git and GitHub.  
It demonstrates how to create a remote repository, clone it to a local machine, and verify the repository status using Git commands.

## Purpose
The purpose of this repository is to:
Set up a GitHub repository
Clone a remote repository to a local environment
Use basic Git commands such as `git status`
Practice version control workflows

## Tools Used
Git
GitHub
JupyterLab (Terminal)
